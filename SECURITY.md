# Security Policy

## Supported Versions

Auraen OS is currently in development. No stable releases are available yet.

| Version | Supported          |
|---------|--------------------|
| develop | ✅ Active development |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue with the `security` label.

Do not disclose security vulnerabilities publicly until they have been addressed by the maintainers.
