# ISO Build Directory

This directory contains the archiso profile for building Auraen OS ISO images.

See [docs/building.md](../docs/building.md) for build instructions.

## Structure

- `airootfs/` - Root filesystem overlay configuration
- `efiboot/` - EFI boot files
- `grub/` - GRUB bootloader configuration
- `syslinux/` - Syslinux bootloader configuration
- `profiledef.sh` - Profile definition for mkarchiso
