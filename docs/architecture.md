# Auraen OS Architecture

## Base System
- Arch Linux
- Built using archiso

## Display System
- Wayland
- GNOME

## Package Management
- pacman
- yay (AUR support)

## Core System Components
- Linux Kernel
- systemd

## Developer Environment
- git, nodejs, npm
- python, pip
- docker

## Terminal Environment
- zsh
- oh-my-zsh

## UI/UX
- Custom GNOME theme (Auraen theme)
- Minimal, clean, developer-focused layout

## Build System
- archiso

## Objective
Create a bootable ISO with all tools preinstalled and ready for development.
